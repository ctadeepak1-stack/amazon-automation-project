# Amazon Automation Project

## 📌 Overview
This project automates Amazon website using Selenium WebDriver with Cucumber BDD and TestNG framework.

## 🛠️ Tech Stack
- Java
- Selenium WebDriver
- Cucumber BDD
- TestNG

## 📂 Framework Design
- Page Object Model (POM)
- Hooks for setup and teardown
- Explicit waits for stability

## ✅ Test Scenario
- Open Amazon website
- Search for product
- Select first product
- Add product to cart

## ▶️ How to Run
Run the TestRunner class.

## 📊 Reporting
HTML report generated in `target/report.html`
